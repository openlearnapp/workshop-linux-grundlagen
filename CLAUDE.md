# Workshop: Linux Basics

## Purpose

IT workshop for absolute beginners — from the first terminal command to a productive Linux workflow. No prior Linux experience required.

## Target Audience

- **Level**: Absolute beginner
- **Prior knowledge**: None — starts with "what is a terminal?"
- **Goal**: Work confidently in a Linux shell, navigate the filesystem, manage files

## Structure

- **12 lessons** building up from basics
- **Interface languages**: Deutsch, English, Farsi, Arabic
- **Teaching language**: matches interface language
- **Features**: hands-on commands, filesystem examples

## Labels

`IT`

## Conventions

- Commands are shown in `q` (what to type)
- Explanations in `a` (what it does)
- Lessons build on each other — linear progression
- Learners should try every command in their own terminal

## Development

```bash
# Generate audio (Edge TTS) for each interface language
bash generate-audio.sh deutsch/linux-basics
bash generate-audio.sh english/linux-basics
bash generate-audio.sh farsi/linux-basics
bash generate-audio.sh arabic/linux-basics
```
