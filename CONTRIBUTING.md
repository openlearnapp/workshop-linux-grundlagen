# Contributing

Danke für dein Interesse, diesen Workshop zu verbessern!

## Workshop testen

1. Öffne https://open-learn.app
2. Klicke "Workshop starten" auf der [Landing Page](https://open-learn.app/workshop-linux-grundlagen/)
3. Teste Lektionen, Assessments und Navigation

## Fehler melden

Erstelle ein [GitHub Issue](https://github.com/openlearnapp/workshop-linux-grundlagen/issues) mit:
- Sprache und Lektion
- Was erwartet wurde vs. was passiert ist
- Screenshot (wenn möglich)

## Inhalte verbessern

1. Fork dieses Repository
2. Erstelle einen Branch (`fix/lesson-3-typo`)
3. Ändere die betroffene `content.yaml`
4. Erstelle einen Pull Request

## Regeln

- Alle 4 Sprachen (DE/EN/FA/AR) müssen synchron bleiben
- `rel`-IDs nie ändern
- Lektionsreihenfolge nie ändern
- Assessments im `correct: true` Format
